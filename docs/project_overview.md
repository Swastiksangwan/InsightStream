
# Project Overview

## Project Name
InsightStream

## Project Idea
InsightStream is a data-driven entertainment decision platform for movies and series.

It helps users decide what to watch by combining:
- content details
- ratings from different sources
- normalized scores
- simplified review insights
- watch tracking
- recommendations

## Final Project Direction
The platform is being built as a data-analytics-focused site, not a public review or discussion platform for now.

Core focus:
- data aggregation
- rating normalization
- simplified decision support
- clean UI
- watched and watch later tracking
- future recommendations

## MVP Features Finalized
- homepage with grouped content
- search and discovery
- title detail page
- ratings from different sources
- normalized rating display
- unified score
- review summary and verdict
- watched
- watch later

## Tech Stack Finalized
- Frontend: Next.js, React, Tailwind CSS
- Backend: Python, FastAPI
- Database: PostgreSQL
- Analytics: Pandas, NumPy, scikit-learn, NLTK/spaCy
- External API: TMDb

## Work Completed So Far
- starter project structure created
- GitHub repository created and updated
- PostgreSQL database created
- MVP database schema finalized
- core tables created
- starter data tested successfully

## Next Step
Connect FastAPI backend to PostgreSQL and create the first working API endpoints.
